C++ Introduction

C++ là ngôn ngữ đa nền tảng có thể được sử dụng để tạo các ứng dụng hiệu suất cao.

C++ được phát triển bởi Bjarne Stroustrup, như một phần mở rộng của ngôn ngữ C.

C++ cung cấp cho người lập trình khả năng kiểm soát cao đối với tài nguyên hệ thống và bộ nhớ.

Ngôn ngữ này đã được cập nhật 4 lần chính vào năm 2011, 2014, 2017 và 2020 thành C++11, C++14, C++17, C++20.

C++ là một trong những ngôn ngữ lập trình phổ biến nhất thế giới.

C++ có thể được tìm thấy trong các hệ điều hành, Giao diện người dùng đồ họa và các hệ thống nhúng ngày nay.

C++ là ngôn ngữ lập trình hướng đối tượng mang lại cấu trúc rõ ràng cho các chương trình và cho phép sử dụng lại mã, giảm chi phí phát triển.

C ++ có thể mang theo được và có thể được sử dụng để phát triển các ứng dụng có thể thích ứng với nhiều nền tảng.

C++ rất thú vị và dễ học!

Vì C++ gần với C , C# và Java nên lập trình viên dễ dàng chuyển sang C++ hoặc ngược lại.

C++ Getting Started

Để bắt đầu sử dụng C++, bạn cần hai thứ:

  • Một trình soạn thảo văn bản, như Notepad, để viết mã C++
  • Trình biên dịch, như GCC, để dịch mã C++ sang ngôn ngữ mà máy tính sẽ hiểu

Có nhiều trình soạn thảo văn bản và trình biên dịch để lựa chọn. Trong hướng dẫn này, chúng tôi sẽ sử dụng một IDE (xem bên dưới).

IDE cài đặt C++

Một IDE (Môi trường phát triển tích hợp) được sử dụng để chỉnh sửa và biên dịch mã.

Các IDE phổ biến bao gồm Code::Blocks, Eclipse và Visual Studio. Tất cả đều miễn phí và chúng có thể được sử dụng để chỉnh sửa và gỡ lỗi mã C++.

=>>IDE mình hay xài đó là Visual Studio Code. Vì nó có nhiều tiện ích hỗ trợ cho nhiều ngôn ngữ lập trình rất bổ ích mình khuyên ae nên test thử nha.

Bắt đầu thử nghiệm nào

Hãy tạo tệp C++ đầu tiên cho mình nào.

Mở Visual Studio Code vài đi tới File > New File. Nhập tên file sau đó bấm Enter và lưu vào Folder cần lưu nha.

Để biên dịch mã C++, bạn cần một trình biên dịch. Trong hướng dẫn này, chúng tôi sẽ sử dụng GCC.

Bắt đầu viết mã C++ nào. Mình sẽ tạo file myfirstprogram.cpp (Đuôi .cpp định dạng tệp C++ Nôm na là như vậy).

#include <iostream>
using namespace std;
int main() {
  cout << "Hello World!";
  return 0;
}

Nào đến phần chạy code thôi nào.

Để chạy được chương trình này thì Click vào cái hình tam giác ở trên góc phải hoặc tổ hợp phím Ctrl + Alt + N.

Kết quả nè:

Xin chúc mừng bro ! Bây giờ bạn đã viết và thực thi chương trình C++ đầu tiên của mình.

C++ Syntax

Dưới đây mình sẽ giải thích từng dòng code nhé.

Dòng 1: #include <iostream> là thư viện tệp tiêu đề cho phép chúng ta làm việc với các đối tượng đầu vào và đầu ra, chẳng hạn như cout(được sử dụng trong dòng 5). Các tệp tiêu đề thêm chức năng cho các chương trình C++.

Dòng 2: using namespace std có nghĩa là chúng ta có thể sử dụng tên cho các đối tượng và biến từ thư viện chuẩn.

Dòng 3: Một thứ khác luôn xuất hiện trong chương trình C++, đó là int main(). Đây được gọi là một chức năng. Bất kỳ mã nào bên trong dấu ngoặc nhọn của nó {} sẽ được thực thi.

Dòng 5: cout (phát âm là "see-out") là một đối tượng được sử dụng cùng với toán tử chèn ( << ) để xuất/in văn bản. Trong ví dụ của chúng tôi, nó sẽ xuất ra "Xin chào thế giới".

Lưu ý: Mọi câu lệnh C++ đều kết thúc bằng dấu chấm phẩy ;.

Lưu ý: Phần thân của int main() cũng có thể được viết là:
int main () { cout << "Hello World! "; return 0; }

Hãy nhớ rằng: Trình biên dịch bỏ qua khoảng trắng. Tuy nhiên, nhiều dòng làm cho mã dễ đọc hơn.

Dòng 6: return 0 kết thúc hàm main.

Dòng 7: Đừng quên thêm dấu ngoặc nhọn đóng } để thực sự kết thúc chức năng chính.

Không dùng Namespace

Bạn có thể thấy một số chương trình C++ chạy mà không có thư viện namespace std . Dòng using namespace std này có thể được bỏ qua và thay thế bằng std từ khóa, theo sau là :: toán tử đối với một số đối tượng:

#include <iostream>
int main() {
  std::cout << "Hello World!";
  return 0;
}

Kết quả vẫn là: Hello World! nhé

C++ Output (In ra ngoài màn hình)

Đối tượng Cout cùng với toán tử << được sử dụng để in giá trị văn bản

Example:

std::cout << "Values";

Bạn có thể thêm bao nhiêu Cout tùy thích

Example:

#include <iostream>
int main() {
  std::cout << "Hello World! 1";
  std::cout << "Hello World! 2";
  std::cout << "Hello World! 3";
  return 0;
}
New Lines

Để chèn thêm 1 dòng mới ta sử dụng ký tự \n

#include <iostream>
int main() {
  std::cout << "Hello World! 1\n";
  std::cout << "Hello World! 2\n";
  std::cout << "Hello World! 3";
  return 0;
}

Kết quả:

Hello World! 1

Hello World! 2

Hello World! 3

Cách khác: Thêm cái này vào cuối dòng endl

#include <iostream>
using namespace std;
int main() {
  cout << "Hello World! 1" << endl;
  cout << "Hello World! 2" << endl;
  cout << "Hello World! 3";
  return 0;
}

Kết quả:

Hello World! 1

Hello World! 2

Hello World! 3

Ký tự xuống dòng ( \n ) được gọi là một escape sequence và nó buộc con trỏ thay đổi vị trí của nó về đầu dòng tiếp theo trên màn hình. Điều này dẫn đến một dòng mới.

Ví dụ về các escape sequence
Escape Sequence Description
\t Thêm khoảng trắng Tab
\\ Chèn ký tự (\)
\" Chén ký tự (")

C++ Comments (Chú thích)

Nhận xét có thể được sử dụng để giải thích mã C++ và làm cho nó dễ đọc hơn. Nó cũng có thể được sử dụng để ngăn chặn việc thực thi khi kiểm tra mã thay thế. Nhận xét có thể là dòng đơn hoặc nhiều dòng.

Single Line Comments

Nhận xét một dòng bắt đầu bằng hai dấu gạch chéo ( // ).

Bất kì văn bản nào ở phía sau // sẽ bị trình biên dịch bỏ qua(sẽ không được thực thi).

Ví dụ

#include <iostream>
int main() {
  std::cout << "Hello World!"; // This is a comment
  return 0;
}

Kết quả:

Hello World!

Multi Line Comments

Nhận xét nhiều dòng bắt đầu bằng /* và kết thúc bằng */.

Bất kỳ văn bản nào giữa /* và */ sẽ bị trình biên dịch bỏ qua:

#include <iostream>
int main() {
  /* ABCXYZ
  ahiahiahihihiiihi */
  std::cout << "Hello World!"; /* This is a comment */
  return 0;
}

Kết quả:

Hello World!

C++ Variables

Biến là các thùng chứa để lưu trữ các giá trị dữ liệu.

Trong C++, có nhiều loại biến khác nhau (được định nghĩa bằng các từ khóa khác nhau), ví dụ:

  • int - lưu trữ số nguyên (số nguyên), không có số thập phân, chẳng hạn như 123 hoặc -123
  • double - lưu trữ số dấu phẩy động, với số thập phân, chẳng hạn như 19,99 hoặc -19,99
  • char - lưu trữ các ký tự đơn, chẳng hạn như 'a' hoặc 'B'. Giá trị char được bao quanh bởi dấu nháy đơn
  • string - lưu trữ văn bản, chẳng hạn như "Xin chào thế giới". Giá trị chuỗi được bao quanh bởi dấu ngoặc kép
  • bool - lưu trữ các giá trị với hai trạng thái: đúng hoặc sai
Creating Variables

Cú pháp:

<type> <variable_name> = <value>;

Hoặc:

<type> <variable_name>;

Trong đó:

  • type là kiểu dữ liệu trong C++ (int, double,...)
  • variable_name là tên biến, có thể đặt tên biến là: x. Có thể xem cách đặt tên biến tại đây

Ví dụ:

Tạo biến kiểu int có tên là myNum gán giá trị 15

int myNum = 15;
cout << myNum;

Kết quả:

15

Bạn cũng có thể khai báo một biến mà không gán giá trị và gán giá trị sau:

int myNum;
myNum = 15;
cout << myNum;

Kết quả:

15

Lưu ý: nếu bạn gán một giá trị mới cho một biến hiện có, nó sẽ ghi đè lên giá trị trước đó:

Xem ví dụ dưới đây để hiểu hơn nhé:

int myNum = 15;
myNum = 10;
cout << myNum;

Kết quả:

10

Khai báo nhiều biến

Để khai báo nhiều hơn một biến cùng loại , hãy sử dụng danh sách được phân tách bằng dấu phẩy:

int x = 5, y = 6, z = 50;
cout << x + y + z;

Kết quả:

61

Một giá trị cho nhiều biến

Ví dụ:

int x = 5, y = 6, z = 50;
x = y = z = 0;
cout << x + y + z;

Kết quả:

0

Hằng số

Hằng số nó giống như một biến có giá trị cố định và không thể thay đổi.

Cú pháp:

const data_type constant_name = value;

Ví dụ:

const int myNum = 15;
myNum = 10;
cout << myNum;

Kết quả:

15

C++ Input

Bạn đã học cout được sử dụng để xuất (in) các giá trị. Bây giờ chúng ta sẽ sử dụng cin để lấy đầu vào của người dùng.

cin là một biến được xác định trước để đọc dữ liệu từ bàn phím với toán tử trích xuất ( >> ).

Trong ví dụ sau, người dùng có thể nhập một số, số này được lưu trong biến x . Sau đó, chúng tôi in giá trị của x:

int x;
cout << "Enter a number: ";
cin >> x;
cout << "You entered: " << x;

Kết quả:

Enter a number: 10
You entered: 10

Tạo một máy tính đơn giản

Trong ví dụ này, người dùng phải nhập hai số. Sau đó, chúng tôi in tổng bằng cách tính (cộng) hai số:

int x, y;
int sum;
cout << "Enter two numbers to add: ";
cin >> x >> y;
sum = x + y;
cout << "Sum is: " << sum;

Kết quả:

Enter two numbers to add: 10 20
Sum is: 30
